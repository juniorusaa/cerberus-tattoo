## 🇹🇷 Türkçe

**Cerberus Tattoo** — Manisa'nın profesyonel dövme stüdyosunun resmi web sitesi. Sanatçı Emre Canişci tarafından yönetilen stüdyo, özgün ve kalıcı dövme sanatı sunmaktadır.

Site saf HTML, CSS ve JavaScript ile geliştirilmiştir; herhangi bir framework kullanılmamıştır.

## 🇬🇧 English

**Cerberus Tattoo** is the official website of a professional tattoo studio based in Manisa, Turkey. Managed by artist Emre Canişci, the studio offers original and permanent tattoo art.

The site is built with pure HTML, CSS, and JavaScript — no frameworks used.

---

## ✨ Özellikler / Features

- 🎬 **Preloader Animasyonu** — Sayfa yüklenirken etkileyici giriş animasyonu
- 🖱️ **Özel Cursor** — Stüdyonun vibeına uygun özelleştirilmiş fare imleci
- 📱 **Responsive Navbar** — Mobil ve masaüstü uyumlu navigasyon menüsü
- 🖼️ **Galeri** — Tamamlanan dövme çalışmalarının sergi alanı
- 💼 **Hizmetler** — Sunulan dövme hizmetlerinin tanıtımı
- 📬 **İletişim** — Randevu ve iletişim formu / bölümü
- ⚡ **Saf HTML/CSS/JS** — Framework bağımlılığı yok, hızlı ve hafif

## 🚀 Kurulum / Setup

### GitHub Pages üzerinden / Via GitHub Pages

Canlı demo aşağıdaki linkten erişilebilir:

🔗 **[cerberus-tattoo.github.io](https://juniorusaa.github.io/cerberus-tattoo)** *(yakında / coming soon)*

### Yerel çalıştırma / Run Locally

```bash
# Repoyu klonla / Clone the repo
git clone https://github.com/juniorusaa/cerberus-tattoo.git

# Klasöre gir / Enter the directory
cd cerberus-tattoo

# index.html dosyasını tarayıcıda aç / Open index.html in your browser
open index.html
# veya / or: double-click index.html
```

Herhangi bir bağımlılık veya kurulum adımı gerekmez. Doğrudan tarayıcıda açılır.

No dependencies or installation steps required. Open directly in any browser.
